Небольшой pet-проект — чат-сервис для личного использования в кругу друзей.  
По функциональности он отчасти напоминает Discord, но без цели создавать полноценный коммерческий продукт.

Проект был разработан как альтернатива заблокированному в России сервису и на данный момент используется мной и моими друзьями в качестве основного инструмента голосовой связи во время совместных игр.

## Доступный функционал

- приватные пространства
- голосовые каналы
- push-to-talk
- демонстрация экрана
- авторизация через Google, Yandex и GitHub

#### Клиентская часть, а также часть серверной логики разрабатывались с использованием ИИ-агента Codex.

## Используемый стек

- Next.js — клиентская часть
- Electron — desktop-версия
- Janus Gateway — RTC-сервер
- Go — backend
- PostgreSQL — основная база данных
- Redis — кеширование
- Caddy — входной прокси
- Coturn — TURN-сервер
- Docker — контейнеризация
- Grafana + Promtail + Loki — базовый мониторинг

Моя основная специализация — backend-разработка на Python, данный проект использовался в том числе как возможность поработать с другими технологиями и инструментами.

Проект не является production-ready решением и не предназначен для коммерческого использования.
Проект развивается в свободное время и не имеет строгого roadmap.
Возможны баги и нестабильная работа.
