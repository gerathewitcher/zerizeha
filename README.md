Небольшой pet-проект — чат-сервис для личного использования в кругу друзей.  
По функциональности он отчасти напоминает Discord, но без цели создавать полноценный коммерческий продукт.

Проект был разработан как альтернатива заблокированному в России Discord и на данный момент используется мной и моими друзьями в качестве основного инструмента голосовой связи во время совместных игр.

## Доступный функционал

- приватные пространства
- голосовые каналы
- push to talk
- демонстрация экрана
- авторизация через Google, Yandex и GitHub

#### Клиентская часть, а также часть серверной логики разрабатывались с использованием ИИ-агента Codex.  
Моя основная специализация — backend-разработка на Python.

Проект не является production-ready решением и не предназначен для коммерческого использования.  
Возможны баги и нестабильная работа.
